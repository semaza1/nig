NIG Management System

Ikoranabuhanga rigezweho ryo gucunga Ikimina (NIG), rigena imigabane, inguzanyo n'inyungu mu buryo bwizewe.

Fungura Konti Injira

Ibyerekeye Sisitemu

Intangiriro

NIG Management System ni software igenewe gucunga imirimo y'ikimina/ikoperative irimo imigabane, inguzanyo, inyungu, expenses, assets n'amakuru y'abanyamuryango.

Uburyo bwo Kubara Inyungu

Iyi system ikoresha uburyo bwa Reducing Balance mu kubara inyungu z'inguzanyo. Inyungu ibarwa buri kwezi hashingiwe ku mwenda usigaye.

Ibikora

Gukuramo Amafaranga

Iyo umunyamuryango ashaka kuvana amafaranga, system ireba inguzanyo, igabanya % yo gukata.

Raporo

Raporo ya buri muntu n'iz rusange, z'ukwezi, igice cy'umwaka n'umwaka.

Umutekano

Login system n'roles, audit trail, backup na restore.

Abishingira

System igenzura imigabane yabo n'inguzanyo zishingira.
